# markdown 常用语法
### 1. 标题 (Headings)
使用 # 号开头，# 的数量代表标题的级别（最多支持六级）。
```Markdown
# 一级标题
## 二级标题
### 三级标题
```
### 2. 文本样式 (Emphasis)
加粗：使用两个星号 ** 或下划线 __ 包裹。

斜体：使用一个星号 * 或下划线 _ 包裹。

删除线：使用两个波浪号 ~~ 包裹。

```Markdown
**这是加粗文字**
*这是斜体文字*
~~这是删除线内容~~
```
### 3. 列表 (Lists)
无序列表：使用星号 *、加号 + 或减号 - 后加空格。

有序列表：使用数字加英文句点 1. 后加空格。

任务列表：使用 - [ ] (未完成) 或 - [x] (已完成)。

嵌套列表：缩进 2 个或 4 个空格（或一个 Tab）来创建子列表
```
- 主项目1
    - 子项目A
    - 子项目B
- 主项目2
    - 子项目A
    - 子项目B
```

```Markdown
- 无序项 A
- 无序项 B

1. 第一行
2. 第二行

- [x] 已完成任务
- [ ] 待办任务
```
### 4. 链接与图片 (Links & Images)
```
链接：[链接文字](URL "可选标题")

图片：'![替代文字](图片地址 "可选标题") '
```
__例__
```Markdown
[百度](https://www.baidu.com)
![示例图](https://example.com/image.png)
```
### 5. 代码 (Code)
行内代码：使用反引号 ` 包裹。

代码块：使用三个反引号 ``` 包裹，并在第一行指定语言（如 python, js）以开启语法高亮。
```Markdown
<三个反引号> inline code
代码
代码
代码
<三个反引号>
```
__例__
```python
def get_len(a, b)
    len = a + b
    return len 
```

### 6. 引用与分割线 (Quotes & Rules)
引用：使用 > 开头。

分割线：使用三个或以上的星号 ***、减号 --- 或下划线 ___。

```Markdown
> 这是一段引用文字。

---
```
__例__
> jk
---
### 7. 表格 (Tables)
使用竖线 | 分隔列，第二行用减号 - 分隔表头和内容，并可用冒号 : 控制对齐方式。
| 左对齐 | 居中 | 右对齐 |
| :---: | :---: | :---: |
| :--- | :---: | ---: | 
```Markdown
| 姓名 | 年龄 | 职位 |
| :--- | :--: | ---: |
| 张三 | 25 | 开发 (左对齐) |
| 李四 | 30 | 设计 (居中) |
```

### 其他
__转义字符：__
如果你想在文档里显示这些特殊符号本身，可以在符号前加反斜杠 \，例如 \*。

__图形符号：__
使用英文冒号+图形英文，比如你想搞个
___“你已急哭”___，那么使用下面的语法弄一个外星人：
```
:alien 
```
👽

__换行：__
在行尾输入两个空格后回车，或者直接空出一行。
